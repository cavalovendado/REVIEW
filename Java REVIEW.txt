ESTRUTURAS de CONTROLE: Tomando Decisões
Para que o programa tome decisões baseadas em condições, usamos as estruturas if (se) e else (senão). Elas permitem que seue código execute diferentes ações dependendo das circunstâncias.


EXEMPLO PRÁTICO: Verificando maioridade

public class Decisao{
 public statoc void main(String[]args) {
  int idade = 17;

  if (idade >= 18){
   System.out.println("Acesso permitindo.");
  } else {
   System.out.println("Acesso negado.");
   System.out.println("Menor de idade.");
  }
 }
}

O programa avalia a condição idade >= 18. Se for verdadeira, executa o primeiro bloco; caso contrário, executa o bloco else.


Estruturas de Repetição: Loops

Por que usar loops?
Quando você precisa repetir uma ação várias vezes, loops tornam seu codigo mais eficiente e limpo. Em vez de escrever o mesmo comando repretidamente, você define uma estrutura de repetição.

O loop for
O loop mais comum em java. Ele tem três partes:
Inicialização: onde começa a contagem
2. Condição: até quando repetir
3. Incremento: como avançar a cada vez

Exemplo: Contando de 1 até 5

public class Loop{
 public static void main(String[]args){
 // (inicio; condição de parada; incremento)
  for(int i = 1; i <= 5; i++){
   System.out.println("Contagem:" + i);
  }
 }
}
