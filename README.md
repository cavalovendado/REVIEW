 # COMANDOS GITBASH
1 - ls = Lista os arquivos e pastas do diretório atual.

2 - cd documents = Entra na pasta Documents.

3 - ls = Lista o que tem dentro de Documents.

4 - cd (nome da pasta) = Entra na pasta do seu projeto.

5 - code . = Abre a pasta no VS Code.

6 - ls = Confere os arquivos do projeto.

7 - git add = Adiciona os arquivos para serem preparados para o commit.

8 - git config --global user.name "cavalovendado" = Define o nome do autor dos commits.

9 - git config --global user.email "pedrinho.henrique.0208@gmail.com" = Define o e-mail que ficará vinculado aos commits.

10 - git commit -m "comentário" = Salva uma versão do projeto.

11 - git branch -M main = Renomeia a branch principal para main.

12 - git push -u origin main = Envia seu projeto para o GitHub
